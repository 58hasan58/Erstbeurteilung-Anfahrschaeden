# Erstbeurteilung Anfahrschäden an Eisenbahnüberführungen
Diese Handlungsanweisung beschreibt das Vorgehen bei Anfahrschäden an Eisenbahnüberführungen im Zuständigkeitsbereich der DB InfraGO AG im AIM Hamm.


Hinweis:

Diese Web-App ist ein privates Hilfsmittel und steht in keiner Verbindung zur DB InfraGO AG, der Deutschen Bahn AG oder einem ihrer Tochterunternehmen.
